📋 Belize RDT (Relatório Diário de Trabalho)
O Belize RDT é um sistema leve e eficiente de gestão operacional, projetado para sincronizar o planeamento do encarregado com a execução da equipa de campo. Focado em transparência e agilidade, o sistema automatiza a criação de relatórios fotográficos e o monitoramento semanal.

🚀 Funcionalidades Principais
Gestão de Equipa Dinâmica: Painel para adicionar ou remover colaboradores (Configurado inicialmente com: Lucas, Gamarra, Mateus e Luis).

Agenda Semanal: Planeamento de tarefas de segunda a sábado com visualização centralizada para o administrador.

Segurança e Acessos:

Senha inicial padrão 123 com troca obrigatória no primeiro acesso.

Trava de horário: Funcionários só acedem ao sistema dentro da jornada de trabalho.

Execução com Prova Visual: Registo de início/fim de tarefas e campo para anexo de foto/prova do serviço concluído.

Automação de Relatórios:

Alerta de fechamento às 16:40h.

Geração de relatório formatado para WhatsApp com um clique.

Limpeza Inteligente: Apaga o histórico do dia concluído, mas preserva o planeamento dos dias futuros.

🛠️ Como Utilizar
1. Instalação e Acesso
O sistema é uma aplicação Single File (HTML/JS). Não requer servidor ou banco de dados externo, utilizando o LocalStorage do navegador.

Abra o arquivo index.html em qualquer navegador moderno (Recomendado: Google Chrome).

Para usar como aplicativo no telemóvel: Clique nos "três pontinhos" do navegador e selecione "Adicionar ao ecrã principal".

2. Painel do Encarregado (Admin)
Usuário: admin

Senha: admbelize2026

No painel, você pode agendar tarefas para qualquer dia da semana e monitorar o status (Pendente, Executando ou Concluída) de cada funcionário em tempo real.

3. Painel do Colaborador
Usuário: Nome do funcionário (ex: lucas).

Senha: 123 (no primeiro acesso, o sistema exigirá a criação de uma nova).

O funcionário visualiza apenas as tarefas do dia atual. Ele deve clicar em "Iniciar", tirar a foto ao finalizar e clicar em "Concluir".

⚠️ Recomendações Importantes
Dados Locais: Como o sistema guarda os dados no navegador, não limpe o cache/histórico do navegador, ou os dados serão apagados.

Backup: O envio do relatório diário para o WhatsApp funciona como o seu backup oficial de dados.

Fotos: O sistema suporta fotos diretamente da câmara. Recomenda-se fotos em média resolução para manter o desempenho do navegador.
